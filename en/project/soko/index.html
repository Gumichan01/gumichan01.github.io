Oxy_Sokoban

Tools : CodeBlocks, SDL (version 1.2.15).

Duration : 2013-Stopped

Current version : 0.3.9

This game is a remake of the Sokoban game released in 1982. It contains some levels from the original game and original levels I built.

The engine game has been made. The game is no longer in development.
