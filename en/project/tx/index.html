Target Xplosion

Details

Target Xplosion is a shoot'em up video game I have been working on for more than a year. It was developed in C++, and uses the LunatiX.

Game system

The game consists of various levels. On each level you begin with a rank. Each rank is like a difficulty level. 3 ranks are available:

  • A: The hardest rank
  • B: The normal rank
  • C: The easiest rank

According to your score in a level, you can start the next level with a different rank. The rank is assigned at the end of the first level, not before. Enemies have more health points and a better resistance when the rank is higher. But the player his also more powerful, and projectiles are more powerful for the player and enemies.

Controls

  • Arrows: Move the ship
  • W: basic shot
  • X: Missile
  • C: Bomb
  • LEFT SHIFT: move the ship slowly
  • ESCAPE : Quit the level/Game
  • RETURN : continue (in the result screen)
