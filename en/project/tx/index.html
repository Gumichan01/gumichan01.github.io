Target Xplosion

Details

Target Xplosion is a shoot'em up video game I have been working on for more than two years. It was developed in C++, and uses LunatiX

Game system

The game system is classical. Shoot and don't get hit! Some items are available in game:

  • Heal item
  • Shield
  • Homing missiles
  • Bomb
  • Laser

Controls

  • Arrows: Move the ship
  • W: basic shot
  • X: Missile
  • C: Bomb
  • LEFT SHIFT: move the ship slowly
  • ESCAPE : Quit the level/Game
  • RETURN : continue (in the result screen)
