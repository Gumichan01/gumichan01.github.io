LunatiX

LunatiX est une bibliothèque de création de jeu basé sur SDL2. Il est utilisé dans le cadre du développement de Target Xplosion. Il peut être utilisé pour faire un projet libre ou commercial grâce à la licence zlib/libpng.

Cette bibliothèque contient différents modules :

  • LX_Graphics : Un module d'affichage 2D reposant sur SDL_Image
  • LX_TrueTypeFont : Un module permettant d'afficher du texte. Il est basé sur SDL_TTF
  • LX_Mixer : module audio
  • LX_Physics : Primitives de fonction pour la detection de collision et le déplacement d'objets
  • Et bien d'autres...

Le moteur fonctionne sous Windows et Linux et est accessible içi.
