LunatiX Engine

Le LunatiX engine est un moteur de jeu basé sur SDL2. Il est utilisé dans le cadre du développement de Target Xplosion. Il peut être utilisé dans le cadre d'un projet libre ou commercial grâce à la licence zlib/libpng.

Ce moteur possède différents modules et bibliothèques :

  • LX_Graphics : Un module d'affichage 2D reposant sur SDL_Image
  • LX_TrueTypeFont : Une biblothèque permettant d'afficher du texte. Il est basé sur SDL_TTF
  • LX_Mixer : Il permet de jgérer différent fichiers audio
  • LX_Physics : Le moteur physique

    Le moteur possède aussi des modules auxiliaires :

  • LX_Device : Permet de gérer diverses manettes
  • LX_FileIO : Une bibliothèque d'entrée/sortie sur les fichiers
  • LX_MSGBox : Un module d'affichege de boite de dialogue
  • LX_ParticleEngine : Un mini moteur définissant un système de particule
  • LX_Random : Gère différent types de générateurs de nombres aléatoires
  • LX_SystemInfo : Un module d'information système

    • Le moteur fonctionne sous Windows et Linux et est accessible içi.
