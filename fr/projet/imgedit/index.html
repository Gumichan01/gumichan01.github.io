ImgEdit

Durée du projet : 4 semaines

ImgEdit est un mini-projet développé lors des différents travaux pratiques de programmation Java consacrés à l'imagerie. Ce mini-projet, réalisé seul durant mes deux années de DUT Informatique, avait surtout pour but de comprendre le fonctionnement d'un logiciel de traitement d'image.

ImgEdit est sous licence WTFPL ( DO WHAT THE FUCK YOU WANT TO PUBLIC LICENSE )

NB: Le projet étant très vieux (et pas remis à jour) et réalisé à l'époque où j'étais un piètre programmeur, il est fort probable que le code source soit très sale.
